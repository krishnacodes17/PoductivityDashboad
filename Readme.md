# Productivity Dashboard

A modern and responsive Productivity Dashboard built using HTML, CSS, and JavaScript. This project helps users manage daily tasks, organize schedules, improve focus, and track productivity through an intuitive dashboard interface.

## Features

- User Authentication (Login & Signup)
- Dashboard Overview
- Todo List (CRUD)
- Daily Planner
- Pomodoro Timer
- Weather Widget
- Motivational Quotes
- Finance Tracker
- Dashboard Statistics
- LocalStorage Data Persistence
- Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
- Font Awesome
- Remix Icons

## Project Goal

The goal of this project is to build an industry-level productivity dashboard using only Vanilla JavaScript while learning DOM manipulation, CRUD operations, LocalStorage, modular architecture, and responsive UI development.
